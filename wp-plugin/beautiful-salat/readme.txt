== Description ==

Salam Alaykoum,

This is a Salat plugin from https://www.freeislamictemplates.com/:


= Features =

- Prayer times widget for your website
- Display daily, weekly, monthly, or yearly tables
- Uses Islamic Finder data
- Over 20,000 cities
- Automatically updates
- Customize colors, title, location

= Working On =

- Add international support
- Add shortcodes


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/beautiful-salat` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
3. Add widget Beautiful Salat
4. Edit widget to change location
